An LLM caching and deduplication layer.

Deliverables:
- package.json
- src
- tests
- examples
- docs
- README.md
- CONTRIBUTING.md
- LICENSE
- CI workflow

Cache and deduplicate expensive model calls, with exact-match caching, TTLs, adapters, and clear cost-saving value.
